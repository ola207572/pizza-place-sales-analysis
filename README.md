# 🍕 Pizza Place Sales Analysis

## 📖 Project Overview
This project analyzes sales data from a pizza restaurant using Python and the Pandas library. The objective is to explore customer ordering patterns, identify the best-selling pizzas, analyze revenue, and generate insights that can support business decision-making.

## 🎯 Objectives
- Clean and prepare the sales dataset.
- Explore the relationships between different datasets.
- Analyze total orders and revenue.
- Identify the best-selling pizza categories and sizes.
- Discover sales trends and customer preferences.
- Present findings using tables and visualizations.

## 📂 Dataset
The project uses four datasets:

- `orders.csv`
- `order_details.csv`
- `pizzas.csv`
- `pizza_types.csv`

These datasets are stored in the `data/` folder.

## 🛠️ Tools and Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## 📊 Analysis Performed
The project includes:
- Data loading and inspection
- Data cleaning
- Merging multiple datasets
- Exploratory Data Analysis (EDA)
- Revenue analysis
- Best-selling pizzas
- Pizza category analysis
- Pizza size analysis
- Data visualization

## 📈 Key Insights
- Identified the highest-selling pizzas.
- Calculated total revenue generated.
- Analyzed pizza sales by category and size.
- Explored customer ordering patterns.
- Created charts to communicate business insights.
